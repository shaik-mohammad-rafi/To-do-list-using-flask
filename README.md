# To-do-list-using-flask
This project is a simple and efficient to-do list application built using HTML, CSS, and Python. It allows users to add, view, and manage their daily tasks through a clean and responsive interface. The backend handles basic task management while the frontend keeps everything minimal and easy to use. This project is ideal for beginners exploring full-stack development.

Features:
Add and manage tasks
Clean and responsive UI
Simple server-side processing using Python
Easy-to-understand code structure

Technologies Used:
HTML
CSS
Python (Flask or basic backend logic)
